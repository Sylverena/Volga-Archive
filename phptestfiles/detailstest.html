Bypass Codes

Click 'Users' on the left hand sidebar as shown:

Search for the user by name or username using the search box as shown:

Scroll to section labeled 'Bypass Code'

Single use

Click Generate Bypass Code *Note this will expire after 1 use OR in 60 minutes*

Click the copy icon next to the hidden bypass code to copy the code.

Click the view (eye) icon to view the passcode

Multiple use/long term

Click the link labeled 'change options'

Click the radio button labeled 'After ___ minutes', then the radio button labeled 'Unlimited'. Then, click 'Generate bypass code'

--- NOTE: The maximum time a bypass code can be active is 12 hours, or 720 minutes ---

Click the copy icon next to the hidden bypass code to copy the code.

Click the view (eye) icon to view the passcode

Adding a user

To add users to Duo, they must be added into the 'DUO User' group in Active Directory first.

Open Active directory, and enter 'DUO' in the search box, then open the 'DUO Users' group by double clicking it

Click the 'Members' tab, and then click 'Add'

Type in the user's name as shown and click 'Check names.' Be sure to select the proper username in the case of multiple accounts

Click on the desired name, then click 'OK'

Next, go to the Duo Admin portal, and click 'Users'

Then, Click 'directory sync' on the top right hand corner.

Click 'Active Directory'

Enter the username(s) you want to add, then click 'Sync Users'

The user is now added to Duo and can sign in to the Device management portal

Adding a device

Click on the 'Users' link on the sidebar

Search for the user you wish to add a device for.

Once on the user page, scroll to the applicable device you wish to add and follow the prompts from there.

Unlocking a user

Only unlock a user if requested by the user themselves, or if there's a ticket related to it.

Click on the 'Users' link on the sidebar

Clicked the 'Locked out' button as shown

On the user's page, select the 'Active' radio button to unlock them as shown. Then, click 'save changes'
